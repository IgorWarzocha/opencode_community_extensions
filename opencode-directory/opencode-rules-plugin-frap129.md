# OpenCode Rules Plugin (frap129)

**Link:** https://github.com/frap129/opencode-rules

**Summary:** A work-in-progress plugin that automatically discovers and injects Markdown-based rule files into OpenCode's AI system prompts, similar to Cursor's rules system. Features include:

- Automatic rule file discovery from standard directories
- Integration into AI agent prompts
- Customizable agent behavior through rules
- Markdown-based rule format
- Similar functionality to Cursor's rules system
- Project-specific and global rule support

This plugin allows users to define custom rules and behaviors for OpenCode agents, providing a structured way to guide AI behavior based on project requirements and personal preferences.
