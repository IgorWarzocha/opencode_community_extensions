# OpenCode MCP Tool

**Link:** https://github.com/frap129/opencode-mcp-tool

**Summary:** A Model Context Protocol (MCP) server that allows AI assistants to interact with the OpenCode CLI tool. Features include:

- MCP server implementation for OpenCode CLI
- Enables AI assistants to control OpenCode programmatically
- Standardized protocol integration
- Remote control capabilities
- Tool bridging between AI assistants and OpenCode
- Enhanced automation possibilities

This tool allows other AI assistants (like Claude) to interact with and control OpenCode through the standardized MCP protocol, creating powerful multi-agent workflows.
