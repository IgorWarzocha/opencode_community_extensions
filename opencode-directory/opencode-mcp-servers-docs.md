# OpenCode MCP Servers Documentation

**Link:** https://opencode.ai/docs/mcp-servers/

**Summary:** Official documentation for using Model Context Protocol (MCP) servers with OpenCode to add external tools and capabilities. Features include:

- MCP integration for external tool connections
- Support for both local and remote MCP servers
- Database access capabilities
- API integrations with third-party services
- Standardized protocol for tool extensions
- Enhanced AI assistant capabilities

MCP servers allow OpenCode to integrate with external systems, databases, and APIs, significantly expanding its capabilities beyond the built-in tools through a standardized protocol.
