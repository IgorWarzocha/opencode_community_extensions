# OpenCode Neovim Plugin (sudo-tee)

**Link:** https://github.com/sudo-tee/opencode.nvim

**Summary:** An alternative Neovim plugin that provides a bridge between Neovim and the OpenCode AI agent. Features include:

- Chat interface within Neovim
- Editor context capture (current file, selections)
- Integration with OpenCode's AI capabilities
- Terminal-based AI assistance workflow
- Lightweight and focused implementation

This plugin offers Neovim users another option for integrating OpenCode, with a focus on providing a clean chat interface while maintaining awareness of the current editing context.
