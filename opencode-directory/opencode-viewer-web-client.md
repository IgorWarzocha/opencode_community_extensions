# OpenCode Viewer Web Client

**Link:** https://github.com/nogataka/opencode-viewer

**Summary:** A full-featured web client for OpenCode with comprehensive session management capabilities. Features include:

- Web-based interface for OpenCode sessions
- Session resumption and inspection
- Workspace filtering and management
- Project organization by name or path
- Full-featured client experience
- Alternative to terminal interface

This web client provides a complete browser-based solution for managing and interacting with OpenCode sessions, offering better session management and organization capabilities than the default terminal interface.
