# OpenCode Observability Platform (OCSight)

**Link:** https://github.com/heyhuynhgiabuu/ocsight

**Summary:** OpenCode observability platform with real-time analytics, MCP server, and markdown reports. Features include:

- Real-time usage analytics and monitoring
- MCP server integration
- Markdown-based reporting
- Performance tracking
- Usage cost monitoring
- Observability dashboard

This platform provides comprehensive monitoring and analytics for OpenCode usage, helping users understand their AI coding patterns, costs, and performance metrics.
