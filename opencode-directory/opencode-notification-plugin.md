# OpenCode Notification Plugin

**Link:** https://gist.github.com/rstacruz/024a1d798c315c7f1c4607c7cf433a4e

**Summary:** A desktop notification plugin for OpenCode that shows notifications with summaries of what the agent last said. Features include:

- Desktop notifications for OpenCode responses
- Summary of agent's last communication
- Easy setup with AGENTS.md configuration
- TypeScript implementation
- Visual feedback for long-running AI tasks

This plugin enhances the OpenCode experience by providing desktop notifications, allowing users to multitask while waiting for AI responses and stay informed of progress without keeping the terminal in focus.
