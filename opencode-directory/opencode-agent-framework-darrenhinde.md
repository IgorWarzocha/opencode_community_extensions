# OpenCode Agent Framework (Darren Hinde)

**Link:** https://github.com/darrenhinde/opencode-agents

**Summary:** An AI agent framework for plan-first development workflows with approval-based execution. Features include:

- Plan-first development methodology
- Approval-based execution system
- Multi-language support (TypeScript, Python, Go)
- Structured agent workflows
- Decision-making frameworks for AI assistance
- Integration with OpenCode's core capabilities

This framework provides a structured approach to AI-assisted development, emphasizing planning and approval steps before execution, making it suitable for teams that require more control over AI-generated code changes.
