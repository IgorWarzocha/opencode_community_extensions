# OpenCode Go SDK

**Link:** https://github.com/sst/opencode-sdk-go

**Summary:** Official Go SDK that provides convenient access to the OpenCode REST API from Go applications. Features include:

- Complete Go API wrapper for OpenCode
- Generated with Stainless for consistency
- REST API integration
- Type-safe Go interfaces
- Easy integration into Go projects
- Programmatic access to OpenCode capabilities

This SDK enables Go developers to integrate OpenCode's AI coding capabilities into their applications and services programmatically, leveraging Go's performance and concurrency features.
