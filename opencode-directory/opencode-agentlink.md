# OpenCode AgentLink

**Link:** https://github.com/martinmose/agentlink

**Summary:** A tool designed to sync AI agent configuration files across different AI coding assistants, providing "one file to rule them all" functionality. Features include:

- Synchronization of AGENTS.md files across tools
- Support for multiple AI assistants (OpenCode, Claude Code, etc.)
- Unified configuration management
- Cross-tool compatibility
- Simplified setup for multiple AI tools
- Configuration file standardization

AgentLink solves the problem of managing different configuration files for various AI coding assistants by providing a unified approach to sync and manage agent instructions across OpenCode, Claude Code, and other AI development tools.
