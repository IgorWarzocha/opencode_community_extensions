# TerminalBell Plugin for OpenCode

**Link:** https://x.com/CarlosGtrz/status/1969623801240412281

**Summary:** A simple TypeScript plugin that rings the terminal bell on session idle events or permission prompts to alert users when OpenCode finishes a task. Features include:

- Terminal bell notification system
- Session idle event detection
- Permission prompt alerts
- Task completion notifications
- Simple TypeScript implementation
- User awareness improvements

This plugin solves the problem of not knowing when OpenCode has finished a task or is waiting for user input, providing audible feedback to improve the user experience.
