# OpenCode GitHub Repository

**Link:** https://github.com/sst/opencode

**Summary:** The main GitHub repository for OpenCode, containing the complete source code for the AI coding agent. This repository includes:

- Complete Go-based source code for the CLI application
- Documentation and setup instructions
- Issue tracking and community discussions
- Contribution guidelines
- Release management
- Plugin system implementation
- Configuration examples and templates

The repository is actively maintained with regular updates, bug fixes, and new features. It serves as the central hub for OpenCode development and community collaboration.
