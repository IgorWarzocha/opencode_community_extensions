# OpenCode MCP Tool (frap129)

**Link:** https://github.com/frap129/opencode-mcp-tool

**Summary:** A Model Context Protocol (MCP) server that allows AI assistants to interact with the OpenCode CLI tool. Features include:

- MCP server implementation for OpenCode CLI
- Enables AI assistants to control OpenCode
- Standardized protocol integration
- Remote control capabilities
- Multi-agent workflow support
- Created September 11, 2025

This MCP server allows other AI assistants (like Claude) to interact with and control OpenCode through the standardized MCP protocol, enabling powerful multi-agent workflows and cross-tool integration.
