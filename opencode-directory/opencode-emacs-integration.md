# OpenCode Emacs Integration (opencode.el)

**Link:** https://github.com/colobas/opencode.el

**Summary:** An Emacs package that ports OpenCode tools and prompts to be used in Emacs via gptel (GPT Emacs client). Features include:

- Emacs integration for OpenCode functionality
- gptel compatibility for AI interactions
- Tool porting from OpenCode to Emacs environment
- LSP mode integration
- File operations and language support
- Read and edit capabilities within Emacs

This package brings OpenCode's AI coding capabilities to Emacs users, allowing them to leverage OpenCode tools and workflows within their preferred Emacs environment.
