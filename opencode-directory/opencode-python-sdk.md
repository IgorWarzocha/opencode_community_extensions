# OpenCode Python SDK

**Link:** https://github.com/sst/opencode-sdk-python

**Summary:** Official Python SDK that provides convenient access to the OpenCode REST API from any Python 3.8+ application. Features include:

- Complete Python API wrapper for OpenCode
- REST API integration
- PyPI package distribution
- Type hints and documentation
- Easy integration into Python projects
- Programmatic access to OpenCode capabilities

This SDK enables Python developers to integrate OpenCode's AI coding capabilities into their applications, scripts, and automation workflows programmatically.
