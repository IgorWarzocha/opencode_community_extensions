# OpenCode Themes Documentation

**Link:** https://opencode.ai/docs/themes/

**Summary:** Official documentation for OpenCode theme customization, allowing users to personalize the terminal interface appearance. Features include:

- Multiple built-in theme options
- Terminal theme adaptation
- Custom theme creation and definition
- Color scheme customization
- Visual appearance control
- User preference settings

Users can choose from pre-built themes, automatically adapt to their terminal colors, or create completely custom themes to match their personal preferences and workflow environment.
