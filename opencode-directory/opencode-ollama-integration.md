# OpenCode Ollama Integration

**Link:** https://github.com/p-lemonish/ollama-x-opencode

**Summary:** A step-by-step guide and setup for integrating OpenCode's agentic workflow with Ollama's local models. Features include:

- Complete setup guide for Ollama + OpenCode integration
- Compatibility with Ollama's tool-capable models
- Local AI model usage with OpenCode
- Configuration examples and templates
- Privacy-focused AI coding assistance
- Offline development capabilities

This resource enables developers to use OpenCode with locally hosted AI models via Ollama, providing privacy, offline capabilities, and cost savings compared to cloud-based AI services.
