# OpenCode Skills Plugin (malhashemi)

**Link:** https://github.com/malhashemi/opencode-skills

**Summary:** A hooks-based extension implementing Anthropic's Agent Skills Specification (v1.0) for OpenCode, including support for nested skills and one-to-one parity with external agent protocols. Features include:

- Anthropic Agent Skills Specification v1.0 implementation
- Hooks-based architecture
- Nested skills support
- One-to-one parity with external agent protocols
- Universal skill loading capabilities
- Integration with OpenCode's plugin system

This plugin brings the skills system to OpenCode, allowing users to define and use reusable AI skills that can be loaded on-demand, similar to how Claude Code implements skills.
