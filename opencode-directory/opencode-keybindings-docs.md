# OpenCode Keybindings Documentation

**Link:** https://opencode.ai/docs/keybinds/

**Summary:** Official documentation for OpenCode keyboard shortcuts and custom keybindings, enabling efficient terminal-based workflow. Features include:

- Comprehensive keyboard shortcut reference
- Custom keybinding configuration
- Leader key support for efficient workflows
- Multi-key combination support (ctrl+x then n)
- Workflow optimization through shortcuts
- Personalized key mapping options

Users can customize their keyboard shortcuts to match their preferences and create efficient workflows for common OpenCode operations, reducing reliance on mouse navigation.
