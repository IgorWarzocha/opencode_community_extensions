# OpenCode Nix Flake

**Link:** https://github.com/AodhanHayter/opencode-flake

**Summary:** A Nix flake that automatically stays up-to-date with the latest OpenCode releases through automated workflows that run every 6 hours. Features include:

- Automated updates every 6 hours
- Nix package management integration
- Reproducible builds
- Easy installation on Nix-based systems
- Always-latest OpenCode version
- Zero-maintenance setup

This resource is perfect for Nix users who want to always have the latest version of OpenCode without manual updates, leveraging Nix's declarative package management and automated workflow systems.
