# OpenSkills Universal Loader

**Link:** https://github.com/numman-ali/openskills

**Summary:** Universal skills loader for AI coding agents that brings Anthropic's skills system to all major AI coding tools including OpenCode, Claude Code, Cursor, Windsurf, and Aider. Features include:

- Universal compatibility across AI coding agents
- NPM global installation (npm i -g openskills)
- Anthropic skills specification implementation
- Cross-platform skill sharing
- 562+ stars, active community
- TypeScript implementation

This tool allows users to create and share skills that work across multiple AI coding assistants, providing a standardized way to extend AI capabilities regardless of the specific tool being used.
