# OpenCode Community Discord

**Link:** https://github.com/sst/opencode (Discord link available in README)

**Summary:** Official OpenCode Discord community server for discussions, support, and community collaboration. Features include:

- Real-time chat with OpenCode developers and users
- Community support and troubleshooting
- Feature discussions and feedback
- Plugin and configuration sharing
- Latest announcements and updates
- Community-contributed resources

The Discord server serves as the central hub for the OpenCode community, providing a space for users to get help, share experiences, and connect with the development team.
