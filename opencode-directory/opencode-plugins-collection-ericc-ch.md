# OpenCode Plugins Collection (ericc-ch)

**Link:** https://github.com/ericc-ch/opencode-plugins

**Summary:** A comprehensive monorepo collection of plugins for OpenCode, providing composable utilities for plugin composition, debugging, and notifications. Features include:

- Multiple plugin packages in one repository
- Plugin composition utilities
- Debugging tools for plugin development
- Notification systems
- Community-contributed plugin ecosystem
- Modular plugin architecture

This repository serves as a centralized collection of various OpenCode plugins, making it easier for users to discover and install multiple extensions from a single source.
