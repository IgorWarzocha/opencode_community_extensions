# OpenCode Mobile App (OpenMode)

**Link:** https://github.com/easychen/openMode

**Summary:** A Flutter-based mobile client for OpenCode that provides a seamless interface for interacting with AI coding assistance on mobile devices. Features include:

- Flutter cross-platform development
- Mobile-optimized interface for OpenCode
- Remote control of OpenCode sessions
- Intuitive touch-based interactions
- Android and iOS support
- On-the-go AI coding assistance

This mobile app enables developers to access and control their OpenCode sessions from mobile devices, providing flexibility for AI-assisted coding when away from the desktop.
