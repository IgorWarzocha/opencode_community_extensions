# OpenCode Smart Title Plugin (Tarquinen)

**Link:** https://github.com/Tarquinen/opencode-smart-title

**Summary:** Auto-generates meaningful session titles for OpenCode conversations using AI. Features include:

- AI-powered session title generation
- Automatic conversation naming
- Context-aware title suggestions
- Improved session organization
- Eliminates generic session names
- 4+ stars from community

This plugin solves the problem of sessions having generic or unclear names by using AI to analyze the conversation content and generate descriptive, meaningful titles automatically.
