# OpenCode Dynamic Context Pruning Plugin (Tarquinen)

**Link:** https://github.com/Tarquinen/opencode-dynamic-context-pruning

**Summary:** A Dynamic Context Pruning (DCP) plugin that significantly reduces token usage by removing obsolete tool outputs from conversation context. Features include:

- Automatic context optimization and pruning
- Token usage reduction
- Removes irrelevant old tool outputs
- Maintains conversation relevance
- Performance optimization for long sessions
- Intelligent context management

This plugin addresses one of the key challenges with AI coding assistants - managing context window size and token costs by automatically cleaning up conversation history while preserving important information.
