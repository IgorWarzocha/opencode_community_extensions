# OpenCode TokenScope Plugin

**Link:** https://github.com/ramtinJ95/opencode-tokenscope

**Summary:** Comprehensive token usage analysis and cost tracking plugin for OpenCode AI sessions. Features include:

- Token usage analysis and monitoring
- Cost tracking for multiple AI models
- Usage optimization suggestions
- Session-based analytics
- Real-time token counting
- Cost calculation and reporting

This plugin helps users understand and optimize their OpenCode usage by providing detailed insights into token consumption and associated costs across different AI models and sessions.
