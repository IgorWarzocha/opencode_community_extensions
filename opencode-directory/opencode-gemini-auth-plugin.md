# OpenCode Gemini Auth Plugin

**Link:** https://github.com/jenslys/opencode-gemini-auth

**Summary:** Gemini OAuth Plugin for OpenCode that enables authentication with Google accounts to use existing Gemini plans and quotas. Features include:

- Official Google OAuth authentication
- Integration with existing Gemini subscriptions
- Personal Google account usage
- No additional API key requirements
- Leverages Google's official authentication method
- Updated within the last 6 days

This plugin allows OpenCode users to authenticate with their personal Google accounts to access Gemini models through their existing subscriptions, providing a seamless authentication experience.
