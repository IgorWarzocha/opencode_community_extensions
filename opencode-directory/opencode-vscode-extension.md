# OpenCode VS Code Extension

**Link:** https://marketplace.visualstudio.com/items?itemName=sst-dev.opencode

**Summary:** Official Visual Studio Code extension that integrates OpenCode directly into the VS Code development workflow. Features include:

- Direct integration with VS Code interface
- Terminal-based OpenCode access within the editor
- Seamless workflow between editor and AI assistance
- Context awareness of open files and projects
- Chat interface for AI interactions
- Code suggestions and improvements

This extension allows developers who prefer VS Code to still leverage OpenCode's powerful AI coding capabilities without leaving their familiar IDE environment.
