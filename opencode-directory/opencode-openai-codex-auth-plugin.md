# OpenCode OpenAI Codex Auth Plugin

**Link:** https://github.com/numman-ali/opencode-openai-codex-auth

**Summary:** OAuth authentication plugin for OpenCode that enables personal coding assistance with ChatGPT Plus/Pro subscriptions using OpenAI's official authentication method. Features include:

- Official OpenAI OAuth authentication
- Uses ChatGPT Plus/Pro subscriptions
- No OpenAI API quota consumption
- Personal development use focused
- Same authentication as OpenAI's official Codex CLI
- Simplified authorization process for subscribers

This plugin allows OpenCode users to leverage their existing ChatGPT Plus/Pro subscriptions instead of paying for separate API access, providing cost-effective AI coding assistance.
