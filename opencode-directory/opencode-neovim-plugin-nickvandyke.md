# OpenCode Neovim Plugin (NickvanDyke)

**Link:** https://github.com/NickvanDyke/opencode.nvim

**Summary:** A popular Neovim plugin that integrates the OpenCode AI assistant directly into the Neovim editor. Key features include:

- Editor-aware AI assistance within Neovim
- Streamlined research, code reviews, and requests
- Context awareness of current files and selections
- Chat interface for AI interactions
- Server-Sent Events forwarding as autocmds for automation
- Flexible configuration and API
- 1k+ stars, active community

This plugin is ideal for Vim/Neovim users who want AI coding assistance without leaving their preferred text editor, providing seamless integration between the editor and OpenCode's capabilities.
