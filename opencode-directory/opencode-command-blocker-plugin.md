# OpenCode Command Blocker Plugin

**Link:** https://github.com/knoopx/opencode-plugin-command-blocker

**Summary:** OpenCode plugin to restrict AI access to files and directories using .ignore patterns (gitignore-style). Features include:

- File and directory access control
- Gitignore-style pattern matching
- Security-focused plugin
- Prevents AI from accessing sensitive files
- Customizable ignore patterns
- 3+ stars from community

This plugin provides security controls by preventing OpenCode from accessing sensitive files and directories, giving users fine-grained control over what the AI can see and modify in their codebase.
