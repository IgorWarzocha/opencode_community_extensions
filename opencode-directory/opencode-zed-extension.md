# OpenCode Zed Extension

**Link:** https://github.com/sst/opencode-zed-extension

**Summary:** Official OpenCode extension for the Zed editor, powered by OpenCode's ACP (AI Communication Protocol) support. Features include:

- Direct integration with Zed editor
- ACP protocol implementation
- Native Zed extension architecture
- Seamless workflow within Zed environment
- Official support from OpenCode team
- Recently moved to main sst/opencode repo

This extension brings OpenCode's AI coding capabilities directly into the Zed editor, providing a native integration for Zed users who want AI assistance without leaving their preferred editor.
